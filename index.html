Documentation (Commands)

pr("text")

Print text to the console.

pr("Hello World")

list

List contents of the current directory (cross-platform).

list

make folder <name>

Create a folder.

make folder my_folder

re <filename>

Remove a file (same syntax on Windows/Linux/macOS).

re file.txt

install <language>

Install a programming language using platform package managers.

install python

Error Codes

0  - Success (Positive)
2  - File not found (Positive)
4  - Permission denied (Positive)
6  - Install failed (Positive)
1  - Unknown command (Negative)
3  - Folder not found (Negative)
5  - Invalid syntax (Negative)
99 - Unknown error (Fatal)
